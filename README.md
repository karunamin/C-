# C++

1. C++ 에서의 구조체
    - 구조체의 등장 배경
        1. 연관 잇는 데이터를 하나로 묶으면 프로그램의 구현 및 관리가 용이
        2. 구조체는 연관 있는 데이터를 하나로 묶는 문법적 장치

    - 구조체 변수 선언        
        1. 데이터 뿐만 아니라, 해당 데이터와 연관된 함수들도 함께 그룹을 형성하기 때문에 함수도 하나로 묶는 것에 대해 나름의 가치를 부여할 수 있다.
    
    - 구조체 안에 함수 삽입하기
        1. C++에서는 구조체 안에 함수를 삽입하는 것이 가능하다. 따라서 C++에서는 구조체가 아닌, 클래스라 한다.
    
    - C++에서의 구조체 변수 선언
    - 구조체 안에 enum 상수의 선언

3. 연산
    
    (processing)
    |연산자 종류|연산자|설명|예|
    |:--|:--|:--|:--|
    |산술 연산자|+ - * / %|기본 수학 연산|a + b|
    |증감 연산자|++ --|값 1 증가/감소|i++, --i|
    |관계 연산자|== != > < >= <=|값 비교|a > b|
    |논리 연산자|&&, \|\|, !|논리 연산|!a|
    |대입 연산자|=|값 대입|a = 10|
    |복합 대입 연산자|+= -= *= /= %=|연산 후 대입|a += 5|
    |조건 연산자|?:|삼항 연산자|a>b ? a : b|
    |비트 연산자|& \| ^ ~ << >>|비트 단위 연산|a & b|
    |주소 연산자|&|변수 주소 반환|&a|
    |포인터 연산자|*|포인터 참조|*p|
    |크기 연산자|sizeof|메모리 크기 반환|sizeof(int)|
    |형 변환 연산자|(type), static_cast 등|자료형 변환|(int)3.14|
    |멤버 접근 연산자|. ->|구조체/클래스 접근|obj.x|
    |범위 지정 연산자|::|네임스페이스 지정|std::cout|
    |인덱스 연산자|[]|배열 접근|arr[0]|
    |함수 호출 연산자|()|함수 실행|func()|
    |쉼표 연산자|,|여러 식 순차 실행|a=1, b=2|
